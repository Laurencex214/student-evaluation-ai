The login flowchart illustrates the authentication process. The system supports three roles: Admin, Teacher, and Student, each with a dedicated login portal. Users enter credentials or scan a QR code. If invalid, incorrect input is shown and the user retries. Upon success, the user is authenticated via the database and redirected to their respective dashboard.
